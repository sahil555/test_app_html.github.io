# 🎮 Click the Box Game

A simple browser-based game built using **HTML, CSS, and JavaScript**.
The goal of the game is to click the moving box as many times as possible to increase your score.

---

## 📌 Features

* Simple and lightweight browser game
* Randomly moving target box
* Real-time score tracking
* No external libraries required
* Runs directly in any modern browser

---

## 🕹️ How to Play

1. Open the game in your browser.
2. A **red box** will appear inside the game area.
3. Click the box to increase your **score**.
4. Each time you click the box, it moves to a new random position.

Try to get the highest score possible!

## ▶️ Running the Game

1. Download or clone the repository.

```
git clone https://github.com/yourusername/click-the-box-game.git
```

2. Navigate to the project folder.

```
cd click-the-box-game
```

3. Open the game file in your browser.

```
open game.html
```

or simply double-click `game.html`.
